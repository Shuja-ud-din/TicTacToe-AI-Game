# TicTacToe Game - Dataset Preparation

This project is focused on preparing a dataset for a TicTacToe game. The dataset will be used for training machine learning models or other purposes related to the game.

## Project Structure

- **Dataset Prepration**: Contains scripts and tools for generating and processing the dataset.
- **README.md**: Documentation for the project.
- **requirements.txt**: List of required Python libraries.

## Features

- Generate TicTacToe game states.
- Label game states as wins, losses, or draws.

## Requirements

- Python 3.x
- Required libraries (install via `requirements.txt`).

## Usage

1. Clone the repository:

```bash
git clone <repository-url>
```

2. Navigate to the dataset preparation directory:

```bash
cd "/Dataset Preparation"
```

3. Run the dataset generation script:

```bash
python main.py
```
